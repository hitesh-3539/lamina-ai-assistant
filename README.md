# Lamina AI Assistant

AI-powered voice assistant built using Python, Speech Recognition, Gemini AI, and native macOS text-to-speech.

## Features

- Wake-word detection using "Lamina"
- Voice-based command execution
- AI conversational responses using Gemini AI
- Open websites through voice commands
- YouTube song search automation
- Real-time technology news narration
- Native macOS text-to-speech integration
- Continuous microphone listening system

## Technologies Used

- Python
- SpeechRecognition
- Gemini AI API
- Requests
- Webbrowser
- macOS Speech Engine

## Installation
pip install -r requirements.txt


## Run the Project
python main.py
